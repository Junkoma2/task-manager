# task manager

静かに使い続けられるタスク管理 Web アプリです。

**→ https://junkoma2.github.io/task-manager/**

## なぜ作ったか

タスク管理ツールは多機能なものが多く、管理そのものに時間がかかってしまうことがあります。
このアプリは「やることを書き出して、終わったらチェックする」というシンプルなサイクルだけを大切にしています。毎日使っても疲れないことを優先しました。

## 主な機能

- タスクの追加・編集・削除
- 子タスクへの分解（1階層のみ）
- 期限日の設定と表示（期限切れは赤表示）
- 繰り返しタスク（毎日・毎週・毎月）
- 並び替え（手動ドラッグ・作成日・期限日）
- 完了済みタスクの一括削除
- 完了済みタスクの表示・非表示切り替え
- データのエクスポート・インポート（JSON）
- PWA対応（ホーム画面に追加して使える）
- pull-to-refresh でアプリを更新

## 使い方

1. リスト下部の「+ タスクを追加」をタップして入力する
2. チェックボックスをタップして完了にする
3. タスク行のアイコンから子タスクの追加・編集・削除ができる
4. 削除は2回タップで確認する（誤削除防止）
5. 繰り返しタスクは追加フォームの「繰り返し」で設定する

## データとプライバシー

データはブラウザのlocalStorageにのみ保存されます。外部サーバーへの送信は一切ありません。
ブラウザの「サイトデータを消去」を行うとデータが消えます。定期的なエクスポートバックアップを推奨します。
